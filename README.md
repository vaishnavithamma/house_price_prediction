# House Price Prediction ML Project

## Overview
This project predicts house prices using regression models.

## Dataset Features
- bedrooms, bathrooms
- sqft_living, sqft_lot
- floors, condition
- year, month (from date)

## Feature Engineering
- Handled missing values
- Encoded categorical variables
- Log transformation applied to price

## Models Used
- Linear Regression
- Random Forest
- Gradient Boosting

## Evaluation Metrics
- RMSE
- MAE

## Best Model
Random Forest (based on RMSE)

## How to Run
1. Open notebook
2. Run all cells
3. Use saved model for prediction

## Files
- house_price_prediction.ipynb
- house_model.pkl
- scaler.pkl
